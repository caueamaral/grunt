# Grunt

O Grunt roda em Node.js, então caso você não tenha instalado.

## package.json
Na raiz do projeto, pelo terminal digite "npm install". Ele vai se encarregar de ler as dependências do projeto e criar uma pasta chamada "node_modules". Essa
pasta não precisa ser versionada porque a finalidade do "package.json" é justamente manter as suas dependências em ordem sem a necessidade de você perder
tempo com esses módulos porque eles não são necessários em produção.

## Gruntfile.js
É o arquivo onde você configura as dependências do seu projeto, dizendo como elas irão se comportar.
